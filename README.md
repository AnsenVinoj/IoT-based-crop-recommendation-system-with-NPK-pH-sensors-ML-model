# upload npk_arduino_code - (PATH: `\crop-prediction\npk_arduino_code`)
upload npk_arduino code to arduino via arduino IDE  (connect the USB in COM3 port)

### `#include <ModbusMaster.h>`
### `#include <SoftwareSerial.h>`
